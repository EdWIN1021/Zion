| 捕获方式      | 说明                                      |
| --------- | --------------------------------------- |
| `[x]`     | 值捕获，把外部变量 `x` 的值拷贝到 lambda 内部           |
| `[&x]`    | 引用捕获，把外部变量 `x` 的引用捕获 → lambda 内修改会影响原变量 |
| `[=]`     | 默认值捕获，所有外部变量都按值捕获                       |
| `[&]`     | 默认引用捕获，所有外部变量都按引用捕获                     |
| `[x, &y]` | 混合捕获，x 按值，y 按引用                         |
| `[this]`  | 捕获当前类的 this 指针 → 可以访问成员变量和成员函数          |
