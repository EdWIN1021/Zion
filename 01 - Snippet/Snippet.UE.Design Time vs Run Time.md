# Design Time vs Run Time

- **设计时间 (Design Time)** ✏️
    - 指的就是你打开 **Unreal Editor (虚幻编辑器)**，在里面编辑关卡、拖动模型、调整蓝图的整个过程。
    - 比如，你在场景里放置一个角色，然后运行它的**构造脚本 (Construction Script)** 来生成一些东西，这个时刻就是“设计时间”。

- **运行时间 (Run Time)** ▶️
    - 指的是你点击编辑器里的 **“Play” (播放) 按钮**，或者把你做好的游戏**打包 (Package)** 出来给别人玩的时候。
    - 这时候，游戏逻辑才真正开始，玩家可以控制角色，AI开始行动，游戏规则开始生效。
