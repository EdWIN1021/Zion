- Net Load on Client 
	- Net Load on Client determines whether an Actor is loaded on clients with the level, not whether its state is replicated.
- Replicates 
	- Replicates determines whether an Actor’s state is synchronized from the server to clients.
- Replicate Movement
	- Replicate Movement determines whether an Actor’s movement (location, rotation, velocity) is automatically synchronized from the server to clients.
